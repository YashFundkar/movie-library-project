# Movie Library Project

## Overview
A full-stack Movie Library application developed using Node.js, Express.js, MySQL, HTML, CSS and JavaScript.

## Features
- Add Movies
- View Movies
- Store data in MySQL
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL

## Run the Project

1. Install dependencies:
npm install

2. Start server:
node server.js

3. Open:
http://localhost:5000
