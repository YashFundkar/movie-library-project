# Movie Library Project

## Overview
Movie Library is a full-stack web application that allows users to add and view movies.

## Features
- Add Movie
- View Movies
- Store Data in MySQL
- Responsive Interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap 5
- Node.js
- Express.js
- MySQL

## Project Structure

movie-library
│
├── public
│ ├── index.html
│ ├── style.css
│ └── script.js
│
├── db.js
├── server.js
├── package.json

## How to Run

npm install

node server.js

Open:

http://localhost:5000
